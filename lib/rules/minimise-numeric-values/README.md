## Minimise Decimal Values

**Guideline:** Omit the 0 before a decimal value to save a single character.

```css
/* Instead of this */
margin: 0.5em;

/* Use this */
margin: .5em;
```

## Minimise Zero Values

**Guideline:** Omit the unit for zero values.

```css
/* Instead of this */
margin: 0px;

/* Use this */
margin: 0;
```
